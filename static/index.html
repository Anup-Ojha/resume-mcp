Process

How it works

Four steps from details to inbox-ready PDF.

01

Provide your details

Send your experience, skills, and background as free-form text — no rigid forms or fields to fill.

02

AI writes the resume

Gemini reads your input and writes a complete LaTeX resume with correct sections, strong bullets, and clean formatting.

03

LaTeX compiles the PDF

pdflatex compiles the source to a pixel-perfect PDF — the same tool used by academics and engineers worldwide.

04

Download or send

Receive the PDF via Telegram, download via API, or have it emailed directly to a recruiter on your behalf.

MCP Integration

Works with any AI assistant

ResumeForge runs an MCP (Model Context Protocol) server mounted at /mcp/sse. Connect Claude Desktop, Cursor, or any MCP-compatible client and ask it to create or tailor your resume in natural language.

The MCP server exposes tools for resume generation, JD parsing, bullet enhancement, and PDF management — no separate integration required.

Claude Desktop Cursor Zed Windsurf Any MCP client SSE transport
claude_desktop_config.json
// Claude Desktop MCP config { "mcpServers": { "resume-forge": { "url": "http://localhost:8000/mcp/sse", "transport": "sse" } } } // Then just ask Claude: "Create a resume for a Senior DevOps Engineer with 6 years at AWS and strong Kubernetes and Terraform experience."

REST API

Integrate with anything

All endpoints are documented at /docs. Key routes below.

Method & Route Description
POST/api/create-resume Generate a full resume from free-form text. AI writes the LaTeX, pdflatex compiles it.
POST/api/tailor-smart Tailor an existing or uploaded resume to a job description. Uses saved LaTeX if available.
POST/api/update-resume Update an existing saved resume with natural-language instructions.
POST/api/enhance-bullets Return 3 high-impact ATS-optimised rewrites of a given bullet point.
POST/api/parse-jd Extract skills, experience level, and requirements from a job description file or text.
GET/api/pdfs List all generated PDF resumes with metadata.
GET/api/pdfs/{filename} Download a specific PDF by filename.
GET/api/stats Live counts — total generated, created from scratch, tailored.
GET/api/health System health — confirms LaTeX is installed and ready.
GET/mcp/sse MCP Server SSE endpoint — connect any MCP client here.
